CSS

O QUE É ?

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada na web para controlar a apresentação e o design de documentos HTML (Hypertext Markup Language). Em vez de lidar com o conteúdo e a estrutura de uma página da web, o CSS lida com aspectos visuais, como cores, fontes, margens, espaçamento, posicionamento e outros estilos visuais.

ONDE É USADO ?

Como dito anteriormente, o CSS é uma linguagem de estilo utilizada junto com o HTML, ou seja, permitindo definir cores, fontes, layout e outros aspectos visuais. Aqui estão alguns dos principais locais e contextos em que o CSS é empregado:

- Desenvolvimento Web
- Web Responsiva
- Aplicativos da Web
- Aplicativos Móveis
- Interfaces de Jogos
